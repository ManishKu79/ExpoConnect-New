# ExpoConnect Backend

## Overview
ExpoConnect is a Next Generation Business Expo Management, Networking & Collaboration Platform. This backend serves REST APIs for the mobile application.

## Installation
1. Clone repository.
2. Run `npm install`.
3. Create `.env` file with required variables (see `.env.example`).
4. Run `npm run dev` for development.

## Database
MongoDB Atlas is used. The database schema includes Users, Companies, Events, etc. See `src/models` for details.
